# Models

Thư mục này chứa các trọng số mô hình phục vụ bài toán phát hiện mũ bảo hộ.

- `hemletYoloV8_25epochs.pt`: phiên bản YOLOv8 sau 25 epoch.
- `hemletYoloV8_100epochs.pt`: phiên bản YOLOv8 sau 100 epoch.
- `helmetV5_1.pt`: trọng số tham chiếu từ phiên bản YOLO trước đó.

File mặc định được `main.py` sử dụng hiện tại là `hemletYoloV8_100epochs.pt`.
